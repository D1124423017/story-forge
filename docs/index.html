Story Forge

Codex Agent Skill

把靈感推到完稿的小說鍛造台

story-forge 讓 Codex 成為嚴格小說家、故事醫生、節奏醫生、對話教練與長篇小說完稿教練。

開始使用 查看 GitHub

What it does

不只寫一段好文字，而是推進整本書

小說專案啟動

釐清題材、類型、讀者承諾、主角傷口、核心衝突與結局方向。

長篇路線圖

建立開場、第一轉折、中段反轉、低谷、高潮與結局餘韻。

逐章寫作陪跑

每章先定目標、衝突、揭露、代價與鉤子，再協助寫正文。

卡稿與修訂

診斷卡稿原因，並把修訂拆成結構、角色、節奏、對話與文風階段。

How to use

在 Codex 裡呼叫 story-forge

  1. 在提示中明確寫出 $story-forge 或描述小說、章節、對話、修稿、卡稿等任務。
  2. 提供你的故事材料：題材、角色、章節、世界觀、目前卡住的位置，或既有正文。
  3. 要求具體交付物：啟動簡報、全書路線圖、章節大綱、正文、診斷、連續性台帳或修訂計畫。
  4. 讓它先處理結構與完稿，再進入文風拋光，避免無限重修第一章。

Long-form control

長篇不崩盤的核心：連續性台帳

story-forge 會追蹤角色關係、傷勢、道具、時間線、伏筆、謎團與世界規則，讓後段劇情不會和前段互相打架。

  • 角色知道什麼、隱瞞什麼、欠誰什麼。
  • 傷勢、資源、道具與時間是否合理延續。
  • 伏筆、謎團與結局承諾是否被追蹤。
Story Forge is built for authors who want finished manuscripts, not endless notes.
