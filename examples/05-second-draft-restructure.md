# Example 05: Second-Draft Restructure

Prompt:

```text
用 $story-forge 幫我重構二稿。初稿完整，但前 8 章都是設定，中段沒有反轉，結尾高潮太快。
```

Expected high-standard output:

```text
【目前稿件階段】
完整初稿，應進入結構稿，不要做文風修辭。

【這一輪不要做的事】
不要重寫漂亮句子，不要新增更多世界史，不要先修對話口吻。

【本輪修訂目標】
把設定變成壓力，把中段補成不可回頭的反轉，把高潮擴成選擇與代價。

【必修問題】
前 8 章壓縮成 3 章：每章只保留會製造危機的設定。
中段新增 false victory：主角以為解開規則，其實暴露了自己最珍貴的記憶。
高潮擴成三段：進入、錯誤選擇、最後代價。

【本輪交付物】
新的章節順序表、刪併清單、三個新增轉折、高潮重建 beat。

【完成標準】
每一章結尾都讓下一章更難，且讀者在第 3 章前知道主角會失去什麼。
```
