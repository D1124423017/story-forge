# Example 01: Novel Project Launch

Prompt:

```text
用 $story-forge 啟動一個長篇小說專案。我的靈感是：一個會忘記死者名字的城市。
```

Expected high-standard output:

```text
【小說啟動簡報】

【題材】
一座城市會在每次葬禮後抹去死者姓名，居民只記得「有人離開」，不記得那個人曾經如何改變他們。

【類型與讀者承諾】
文學奇幻 / 懸疑。讀者承諾是：追查一個名字消失的謎，同時看見主角如何對抗集體遺忘。

【主角外在目標】
主角要找回母親真正的名字，阻止下一次大規模遺忘。

【主角傷口】
她害怕自己其實不值得被記住，所以用記錄別人的方式逃避自己的空洞。

【核心衝突】
城市的記憶制度維持和平，但它也讓犯罪、背叛與犧牲都無法被追究。

【反派 / 阻力】
不是單一壞人，而是管理姓名檔案的殯葬署、依賴遺忘活下去的居民，以及主角自己對真相的恐懼。

【結局方向】
苦甜勝利：她找回母親名字，但必須公開一段會傷害整座城市的歷史。

【下一步】
先做全書路線圖，再建立 continuity-ledger，因為本書的核心快感依賴名字、時間線、謎團與伏筆。
```
