# Example 02: Chapter-by-Chapter Companion

Prompt:

```text
用 $story-forge 陪我寫第七章。前情：主角剛背叛朋友換取進城資格，但不知道朋友其實故意讓他背叛。
```

Expected high-standard output:

```text
【本章任務】
讓主角以為自己得到勝利，實際上踏入朋友安排的更大局。

【主角目標】
進城後立刻找到殯葬署的姓名檔案。

【衝突】
通行證有效，但城門官知道他是用背叛換來的，故意讓他公開接受羞辱。

【揭露】
朋友留下的暗號不是求救，而是指向城內第二個入口。

【代價】
主角進城了，但他的背叛變成官方紀錄，之後再也不能假裝自己乾淨。

【結尾鉤子】
他在檔案室看見朋友的名字已經被列入「明日刪除」。

【連續性注意】
背叛未被原諒；朋友的意圖不能太早說破；通行證只能使用一次。
```
