# Example 06: Mature Boundary Repair

Prompt:

```text
用 $story-forge 診斷這個 18+ 黑暗愛情設定：男主角用職權逼女主角留下，故事把這當成浪漫。我要保留危險感，但不要美化傷害。
```

Expected high-standard output:

```text
【成熟題材功能】
你要的是危險吸引與權力壓迫，不是把脅迫包裝成愛。

【目前風險】
若女主角不能拒絕、離開、談條件或反擊，這不是黑暗浪漫張力，而是剝奪 agency。

【權力關係】
男主角的職權是壓迫來源，必須被故事承認為風險與道德負債。

【改法】
保留危險感，但讓女主角有可見選擇：她可以留下，原因不是被逼，而是她用留下換取證據、保護某人、或反過來設局。

【替代 premise】
他以為自己用權力留住她；她其實利用他的傲慢進入禁區。吸引力存在，但每一次靠近都增加她暴露的風險，也增加他失去控制的代價。
```
