# story-forge

A Codex Agent Skill for fiction writing, revision, dialogue coaching, story diagnosis, genre analysis, and narrative design.

## Purpose

story-forge helps Codex act as a strict novelist, story doctor, dialogue coach, genre analyst, and continuity guardian.

## Structure

* SKILL.md: core skill instructions and trigger description
* agents/openai.yaml: Codex UI metadata and invocation prompt
* references/: writing checklists, genre patterns, dialogue guide, revision rubric, and scene templates

## Suggested usage

Use this skill when working on fiction, scripts, game narrative, character arcs, dialogue, scenes, chapters, worldbuilding, or story revision.
