# story-forge

A Codex Agent Skill for fiction writing, revision, dialogue coaching, story diagnosis, genre analysis, narrative design, mature 18+ themes, and original series development.

## Purpose

story-forge helps Codex act as a strict novelist, story doctor, dialogue coach, genre analyst, continuity guardian, mature-content story editor, and original high-concept story development partner.

## Structure

* SKILL.md: core skill instructions and trigger description
* agents/openai.yaml: Codex UI metadata and invocation prompt
* references/: writing checklists, genre patterns, dialogue guide, mature-content rules, revision rubric, scene templates, acceptance tests, and original series-development guidance

## Suggested usage

Use this skill when working on fiction, scripts, game narrative, character arcs, dialogue, scenes, chapters, worldbuilding, story revision, mature 18+ themes, high-concept premises, series bibles, chapter engines, or original alternatives to famous-story inspiration.
